setting up the system:
+Install python

+ install django on system
pip install django

+ install pipenv on system
pip install pipenv

Clone project
git clone https://github.com/Amirhossein-code/restaurant-manager.git

cd into project directory
cd restaurant-manager

then cd into Backend
cd Backend

install virtual env
pipenv install

copy the following into pipfile    
```
[packages]
django = "*"
djangorestframework = "*"
djangorestframework-simplejwt = "*"
djoser = "*"
drf-nested-routers = "*"
django-autoslug = "*"
pillow = "*"
mysql-connector-python = "*"
django-debug-toolbar = "*"
django-filter = "*"
python-dotenv = "*"

[dev-packages]
model-bakery = "*"

[requires]
python_version = "3.10"
```

set up dependencies
pipenv update

Set up Databse 
fisrt create a databse im mysql workbench
CREATE DATABASE example;

Create .env file
configure the variables 
```
DB_NAME=example
DB_USER=root
DB_PASSWORD=
DB_HOST=localhost
DB_PORT=3306
```

Creating databse schema
python manage.py migrate

Create first admin
python manage.py createsuperuser

Runserver 
python manage.py runserver
