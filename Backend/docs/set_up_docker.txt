Fisrt create .env file and place the credentials

```
MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=restaurant
MYSQL_USER=
MYSQL_PASSWORD=

DB_PORT=3306
DJANGO_ALLOWED_HOSTS=localhost 127.0.0.1 [::1]
```

Then
docker-compose up --build

the application should be running now

how to create admin?

first identify the backend container run this in a new terminal
docker ps

Note the container id of backend that is running then run
docker exec -it <container_id> bash

then run this to create admin
python manage.py createsuperuser

enter the credential then cloes terminal with 
cntrl + D
